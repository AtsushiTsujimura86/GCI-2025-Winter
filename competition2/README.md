# コンペ２

**AUC（Area Under the ROC Curve）＝  
「Drafted(1) を Not Drafted(0) より高い確率で予測できた割合」**
---

## ✔ 例が一番わかりやすい

### ● 良い予測（成功 = 1点）
正解：
- A = 1  
- B = 0  

予測：
- A：**0.8**  
- B：0.3  

👉 1のA > 0のB → **成功（1点）**

---

### ● 悪い予測（失敗 = 0点）
正解：
- A = 1  
- B = 0  

予測：
- A：0.4  
- B：**0.7**  

👉 0のBが高い → **失敗（0点）**
---
## ✔ これを「1の選手 × 0の選手」の全組み合わせで計算し、  
成功率（0〜1）にしたものが **AUC**。
